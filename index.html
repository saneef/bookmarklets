Here is a collection of bookmarklets, to speed various tasks on Browser

Force Paste into a Text Field

Have you ever found yourself trying to paste your account number into a text field and the website doesn't allow to paste? This bookmarklet is helps you to forcefully paste into any adamant input fields.

Force Paste into an Text Field ← drag this button to your Bookmarks Bar.

Edit Text on a web page

This is useful for writing/editing copy on a web page. After you have edited the text, you can copy the text and paste into your source code.

Want to try? Click on the button below and click here on this text. Now try typing.

Edit Text ← drag this button to your Bookmarks Bar.

Go to the Google Cache of the page

If a news article page you were looking for has been taken down, give a try on Google Cache. If you are lucky it'll be there.

Go to Google Cache ← drag this button to your Bookmarks Bar.
